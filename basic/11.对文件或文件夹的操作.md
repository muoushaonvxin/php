### php对文件夹的操作方法

<br/>

| 函数名称   | 使用方式                      | 函数作用  |
|-----------|------------------------------|----------|
| mkdir()   | mkdir($directory,0777,true); | 创建文件夹 |
| rmdir()   | rmdir($directory);           | 删除文件夹 |


批量创建文件夹
```php
<?php
$arr = Array('a','b','c','d');
foreach ($arr as $v) {
    $path = './wenjian/'.$v;
    if(file_exists($path)&& is_dir($path)){
        echo $path . "已存在" . "\n";
        continue;
    }

    if(mkdir($path, 0777, true)){
        echo $path . "\n";
    }
}    

// 输出的结果为:
./wenjian/a
./wenjian/b
./wenjian/c
./wenjian/d
```
