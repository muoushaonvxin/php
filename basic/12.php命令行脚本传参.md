
| 函数名称 | 使用方式                            | 函数作用         |
|---------|------------------------------------|------------------|
| $argc   | 终端命令行传参,代码打印该变量显示结果 | 显示传递参数的个数 |
| $argv   | 终端命令行传参,代码打印该变量显示结果 | 显示传递参数的内容按照数组的格式存放 |

```php
<?php
echo "接收到{$argc}个参数";
var_dump($argv);

// 输出运行的结果为:
[root@zhangyz ~]# php test1.php 
接收到1个参数
array(1) {
  [0]=>
  string(9) "test1.php"
}
```
